# Draw
- Draw это приложение в котором можно рисовать что угодно на предоставленном холсте.
- Имеется возможность выбирать толщину кисти, предоставленна схема rgb для любого цвета, можно сохранять все свои картины в папке.
# Зависимости
- express - фреймворк для создания веб-сервисов.
- types/express - пакет с типами для typescript, чтобы express работал
- ts-node -запускает typescript сразу на node.js
# Установка и запуск
- Для того чтобы приложение работало необходимо установить пакеты (команда: npm i), затем загрузить билд (команда: npm run build).
- После этого можно запускать приложение (комманда: npm start).
